
# 无限加载 Infinite loading
"Infinite loading"（无限加载）是一种网页或应用程序设计中常见的交互方式，
其主要特点是在用户滚动页面或浏览内容时，不断加载新的数据，实现无限延伸的页面或列表。
这种设计可以提供更流畅的用户体验，因为用户无需点击“下一页”按钮或刷新页面，
就能够持续地获取新的信息或内容。通常，无限加载被用于长列表或动态内容的展示，以减少页面加载时间和提高用户参与度。

### 在项目中引入 vue-infinite-loading
1、使用npm进行安装：

    npm install vue-infinite-loading

2、在main.js文件里配置

    import InfiniteLoading from 'vue-infinite-loading'
    Vue.use(InfiniteLoading);

3、在项目中使用

在template中使用

    <div class="video-container" v-for="video in videos"
        :key="video.id" @click="jumpToVideoDetail(video)">
    </div>
    <infinite-loading :infinite-id="infiniteId"
                    @infinite="pageListVideos">
    </infinite-loading>

在methods中使用

    pageListVideos($state){
      let params = {
        size:10,
        no:this.currentPage
      }
      videoApi.pageListVideos({params}).then(response => {
        const {list:videos, total:total} = response.data;
        if(videos.length === 0){
          // 已加载所有数据，不再触发加载
          $state.complete();
          return;
        }
        this.videos = this.videos.concat(videos);
        this.total = total;
        this.currentPage++; // 递增当前页码
        $state.loaded(); // 标记加载完成
      }).catch(error => {
        // 捕获异常
        console.error('请求出错:', error);
        $state.complete();
      });
    }
