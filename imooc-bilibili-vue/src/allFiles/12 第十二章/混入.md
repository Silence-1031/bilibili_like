
# 混入
在Vue.js中，你可以使用混入（Mixin）来共享组件中的逻辑。
混入是一种将一些功能复用在多个组件中的方式，允许你在组件中引入混入对象，以重用其中的属性、方法等。


### 1、创建混入对象： 创建一个包含你想要复用的逻辑的混入对象。

    export default {
        data() {
            return {
                defaultAvatar:require('@/assets/defaultAvatar.png'),
            };
        },
        methods{
            //用户密码rsa加密
            encryptPassword(password, publicKey) {
                const encryptStr = new JSEncrypt();
                encryptStr.setPublicKey(publicKey.toString()); // 设置加密公钥
                return encryptStr.encrypt(password); // 进行加密
            },
        },
        computed: {
            //判断用户是否登录
            isUserLoggedIn() {
                let token = localStorage.getItem('token');
                return token && token !== 'null';
            }
        }
    }


### 2、引入混入对象： 在组件中使用 mixins 选项引入混入对象。

    <script>
    
    export default{
        components: {LoginDialog},
        mixins:[userUtils, routerUtils],
    }
    
    </script>
