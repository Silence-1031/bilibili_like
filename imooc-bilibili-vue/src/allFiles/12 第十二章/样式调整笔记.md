
# 样式调整笔记


### 1、opacity
opacity是用于设置元素透明度的属性。它的取值范围是从0到1

    0 表示完全透明，元素不可见。
    1 表示完全不透明，元素完全可见。


### 2、 flex 1 auto
flex: 1 auto; 是用于设置flex布局中子元素的弹性属性的简写形式。
其包括三个值：

    1、flex-grow: 这个值定义了一个弹性项的放大比例，默认为0。如果所有的弹性项都设置为1，它们将等分剩余空间。
    2、flex-shrink: 这个值定义了一个弹性项的收缩比例，默认为1。如果容器空间不足，弹性项将按照其收缩比例减小。
    3、flex-basis: 这个值定义了在分配多余空间之前，弹性项占据的主轴空间。它可以是一个固定的长度（如像素或百分比），也可以是auto，表示由内容决定。

在flex: 1 auto 中，1对应于flex-grow，表示弹性项可以放大，
auto对应于flex-basis，表示弹性项在分配剩余空间之前会占据其内容的空间。


### 使用::v-deep实现样式穿透
样式穿透允许你在子组件中修改父组件的样式，或者在一个组件中修改引入的第三方组件的样式。
::v-deep 是Vue内置的一种样式穿透方式。

      ::v-deep .el-carousel__container{
          height: 100%;
          min-height: 400px;
      }
