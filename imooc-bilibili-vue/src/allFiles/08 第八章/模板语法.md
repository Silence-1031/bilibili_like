
# 模板语法

在Vue中，模板语法有两种，一个是插值语法，另一个是指令语法

### 一、差值语法
差值语法使用双大括号{{ }}将JavaScript表达式嵌入到HTML中，用于动态地显示Vue实例中的数据。

    <div>{{message}}</div>

### 二、指令语法
指令是以v-开头的特殊属性，用于向元素应用特殊行为。
以下是一些常见的Vue指令：

1、v-bind:
用于将元素属性与Vue实例的数据进行单向绑定。
示例：
    
    <a v-bind:href="url">Link</a>

2、v-model:

用于在表单元素和Vue实例的数据之间创建双向绑定。
示例：
    
    <input v-model="message">

3、v-if, v-else-if, v-else:

用于根据表达式的值条件性地渲染元素。
示例：
    
    <p v-if="seen">Now you see me</p>

4、v-for:

用于遍历数组或对象，重复渲染元素。
示例：

    <li v-for="item in items">{{ item.text }}</li>

5、v-on:

用于监听DOM事件，并在触发时执行Vue实例的方法。
示例：

    <button v-on:click="doSomething">Click me</button>
