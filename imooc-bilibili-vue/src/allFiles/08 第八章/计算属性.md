
# 计算属性
在Vue.js中，计算属性（Computed Properties）是一种能够基于响应式数据进行计算的属性。
计算属性的值会被缓存，只有在相关依赖发生变化时才会重新计算。这有助于优化性能，避免不必要的计算。

### 计算属性的声明方式如下：

    export default {
        name: 'HelloWorld',

        computed:{
        
            isUserLoggedIn(){
              return localStorage.getItem('token');
            }
        
        }
    }
