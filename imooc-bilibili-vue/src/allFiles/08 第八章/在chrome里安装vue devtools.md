
# 在chrome里安装vue devtools

### 一、安装Vue Devtools插件
1、打开Chrome浏览器，前往Chrome Web商店（https://chrome.google.com/webstore）

2、在搜索栏中输入 "Vue.js devtools"。

![](../images/在chrome安装vue%20devtools/e7f91951.png)

3、找到Vue.js devtools插件，并点击 "添加到Chrome" 按钮进行安装。

![](../images/在chrome安装vue%20devtools/b158d8df.png)

### 二、确认插件已安装

安装完成后，你将在Chrome右上角看到Vue的图标。如果你看到这个图标，说明安装成功。

![](../images/在chrome安装vue%20devtools/7820c18c.png)

### 三、在Vue应用程序中使用Vue Devtools
打开你的Vue.js应用程序。

在Chrome开发者工具中，你将看到一个名为 "Vue" 的选项卡。点击它，你将能够查看Vue组件树、状态、事件等信息。

![](../images/在chrome安装vue%20devtools/5fbbbf20.png)
