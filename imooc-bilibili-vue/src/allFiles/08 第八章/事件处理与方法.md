
# 事件处理与方法

在Vue.js中，事件处理是一种响应用户交互的方式。可以使用v-on指令来监听事件（如点击按钮、按下键盘等），
并触发相应的Vue实例方法。

### 一、使用v-on监听事件
使用 v-on 指令来监听DOM事件。它可以用在普通的HTML元素上，也可以用在自定义组件上。例如：

    <button v-on:click="handleClick">Click me</button>
简写形式：

    <button @click="handleClick">Click me</button>

上述代码中，v-on:click 监听了按钮的点击事件，并在点击时调用了名为 handleClick 的方法。

### 二、方法定义
在Vue实例中，你可以定义方法，以供事件处理使用。方法定义在 methods 对象中。例如：

    export default {
        name: 'HelloWorld',
        props: {
            msg: String
        },
        data(){
            return {
                registerBtn:'注册按钮',
                loginBtn:'登录按钮',
                account:'account',
                password:'123456'
            }
        },
        methods:{
        
            handleClick(){
              //处理点击事件的逻辑
        
            },
            //其他方法
        }
    }

### 三、传递参数
有时你需要在方法调用时传递参数。你可以使用内联表达式或者调用一个方法来传递参数。例如：

在页面上传递：

    <button v-on:click="handleClick('Hello')">Click me</button>
在方法里接收：

    methods: {

        handleClick(message) {
            console.log(message);
        }
    }
