# 字体控制


### 字体大小：font-size

设置字体的大小。可以使用像素（px）、百分比（%）等单位。

    h1 {
        font-size: 24px;
    }



### 字体粗细：font-weight

设置字体的粗细。常用值有 normal、bold，也可以使用数字值（如 400、700）表示粗细程度。

    p {
        font-weight: bold;
    }


### 字体颜色：color

设置文本颜色。

    p {
        color: #333333;
    }

### 设置文本水平对其方式：text-align

设置文本的水平对齐方式，如左对齐、右对齐、居中等。

    div {
        text-align: center;
    }
