
# 布局控制


### position属性
position 属性用于定义元素的定位方式。以下是几个常用的值：

    1、static (默认值):元素在正常文档流中进行布局。这是默认的定位方式。
    
    2、relative:元素相对于其正常位置进行定位，但仍然保持在文档流中，所以周围的元素不会被调整。
    
    3、absolute:元素脱离正常文档流，在正常文档流中，元素按照其在HTML中的顺序一个接一个地放置。
    块级元素会在页面上独自占据一行，从上到下排列，而内联元素则水平排列在其容器中。
    相对于其最近的已定位祖先元素进行定位，如果没有已定位的祖先，则相对于最初的包含块（通常是 <html> 元素）进行定位。
    
    4、fixed:元素相对于浏览器窗口进行定位，即使页面滚动，元素位置也不会改变。
    
    5、sticky:元素根据用户滚动的位置进行定位。在某个阈值前，它的行为类似于 relative，之后类似于 fixed。

### display 属性
display 属性定义元素应该生成的框类型。以下是一些常见的值：

    1、block:元素将生成一个块级框，会在页面上单独占据一行，并且垂直堆叠。
    
    2、inline:元素将生成一个内联框，不会打断布局，只占据它内容的空间。
    
    3、inline-block:元素表现为内联元素，但可以设置宽度、高度等块级元素的属性。
    
    4、flex:定义一个弹性容器，内部的子元素将使用弹性布局模型进行排列。
    
    5、grid:定义一个网格容器，内部的子元素将使用网格布局模型进行排列。

### align-items 属性
控制 Flex 容器内项目在交叉轴（垂直轴）上的对齐方式。 可能的值：

    1、flex-start: 项目向交叉轴的起始位置对齐。
    
    2、flex-end: 项目向交叉轴的结束位置对齐。
    
    3、center: 项目在交叉轴上居中对齐。
    
    4、baseline: 项目在交叉轴上以基线对齐。
    
    5、stretch: 项目在交叉轴上被拉伸以适应容器。

### justify-content 属性
控制 Flex 容器内项目在主轴（水平轴）上的对齐方式。 可能的值：

    1、flex-start: 项目向主轴的起始位置对齐。
    
    2、flex-end: 项目向主轴的结束位置对齐。
    
    3、center: 项目在主轴上居中对齐。
    
    4、space-between: 项目在主轴上平均分布，首尾项目分别放置在容器的起始和结束位置。
    
    5、space-around: 项目在主轴上平均分布，项目两侧的空白空间相等。
    
    6、space-evenly: 项目在主轴上平均分布，包括首尾项目和容器两侧的空白空间。

### flex布局

Flexbox（弹性盒模型）是CSS中的一种布局模型，它可以更有效地设计复杂的布局，
减少代码量。Flexbox提供了一种沿单一轴（行或列）或两个轴（行和列）分配空间的方式，
允许容器内的元素根据它们的内容或可用空间动态排列和调整大小。

要使用Flexbox，需要将 display: flex;

    .flex-container {
        display: flex;
    }

Flex方向： 使用 flex-direction 属性设置Flex容器的主轴方向。默认值是row，可以设置为column

    .flex-container {
        display: flex;
        flex-direction: row; /* 或 column*/
    }

常见使用模式：

    .flex-container {
        display: flex;
        flex-direction: row;
        align-items: center;
        justify-content: flex-start;
    }


### grid布局

网格布局是一种用于网页布局的二维布局系统，它使得可以更方便地划分页面区域并进行定位。

1、创建Grid容器： 通过将 display: grid; 应用于容器元素，将其设置为Grid容器。

    .grid-container {
        display: grid;
    }


2、定义元素间的距离：

    .grid-container {
        display: grid;
        grid-gap: 20px;
    }

3、定义Grid行和列的规则： 使用 grid-template-rows 和 grid-template-columns 属性来定义Grid的行和列。

    .grid-container {
        display: grid;
        grid-template-rows: 100px 200px; /* 定义两行，高度分别为100像素和200像素 */
        grid-template-columns: 1fr 2fr; /* 定义两列，宽度比例为1:2 */
    }

fr是Grid布局中的一个单位，表示"fraction"（分数）。

它用于指定网格布局中列（grid-template-columns）或行（grid-template-rows）的大小，以分数的形式分配可用空间。

fr单位用于指定相对于其他单位（如像素、百分比等）的分数比例。
一个fr表示可用空间的一个等分， 而多个 fr 则表示这些等分的比例关系。


    .grid-container {
        display: grid;
        grid-gap: 20px;
        grid-template-columns: repeat(5, 1fr); /* 将Grid容器的列划分为5列，每列的宽度都相等 */
    }
