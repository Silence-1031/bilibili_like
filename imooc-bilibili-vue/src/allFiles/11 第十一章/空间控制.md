
# 空间控制


### 外边距 margin
margin 属性用于设置元素的外边距，即元素与其周围元素之间的空间。
margin 可以应用于所有HTML元素，它接受一个或多个值，每个值表示元素在相应的方向（上、右、下、左）的外边距大小。

语法如下：

    margin: [top] [right] [bottom] [left];


此外，还可以使用 margin-top、margin-right、margin-bottom 和 margin-left 分别设置单个方向的外边距。例如：

    .element {
        margin-top: 10px;
        margin-right: 20px;
        margin-bottom: 30px;
        margin-left: 40px;
    }

注意：margin 的值不允许使用负值，但可以使用百分比、像素等单位。


### 内边距 padding
padding 属性用于设置元素的内边距，即元素内容与其边框之间的空间。
padding 可以应用于所有 HTML 元素，它接受一个或多个值，每个值表示元素在相应的方向（上、右、下、左）的内边距大小。

语法如下：

    padding: [top] [right] [bottom] [left];

此外，还可以使用 padding-top、padding-right、padding-bottom 和 padding-left 分别设置单个方向的内边距。
例如：

    .element {
        padding-top: 10px;
        padding-right: 20px;
        padding-bottom: 30px;
        padding-left: 40px;
    }

与 margin 类似，padding 的值也不允许使用负值，但可以使用百分比、像素等单位。



### 圆角 border-radius
border-radius 属性用于设置元素的边框圆角。通过这个属性，你可以使元素的边框呈现为圆角而不是直角。
border-radius 可以应用于所有 HTML 元素。

语法如下：

    border-radius: [top-left] [top-right] [bottom-right] [bottom-left];

每个值可以是一个具体的长度（像素、百分比等），也可以是关键字（如 auto）。

一些示例：

    /* 设置所有角的边框圆角为10像素 */
    .border-radius-10 {
        border-radius: 10px;
    }
    
    /* 设置左上和右下角的边框圆角为10像素，右上和左下角的边框圆角为20像素 */
    .border-radius-10-20 {
        border-radius: 10px 20px;
    }
    
    /* 设置左上角的边框圆角为10像素，右上角的边框圆角为20像素，右下角的边框圆角为30像素，左下角的边框圆角为40像素 */
    .border-radius-10-20-30-40 {
        border-radius: 10px 20px 30px 40px;
    }


border-radius 的值可以为百分比，这表示相对于元素的尺寸。
例如，border-radius: 50%; 将创建一个完美的圆形边框。

### 长宽 
width 和 height 是用于设置元素的宽度和高度的 CSS 属性。这两个属性可以应用于几乎所有 HTML 元素。

width 用于设置元素的宽度。可以使用以下单位：

    像素（px）；
    百分比（%）；
    vw（视窗宽度的百分比）；
    vh（视窗高度的百分比）

height 用于设置元素的高度。类似于width，同样可以使用像素、百分比、vw 和 vh 单位。
